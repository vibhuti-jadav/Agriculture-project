## Troubleshooting Guide

### If pages not indexed:
1. Check robots.txt allows indexing
2. Verify canonical tags correct
3. Check for noindex meta tag
4. Request indexing in Search Console
5. Wait 2-4 weeks

### If rankings not improving:
1. Ensure quality content
2. Build backlinks from relevant sites
3. Improve page speed
4. Increase content depth
5. Ensure keyword relevance

### If low organic traffic:
1. Check keyword competition
2. Verify proper keyword targeting
3. Optimize title/description
4. Build more content
5. Create backlinks
