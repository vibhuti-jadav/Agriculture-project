# Google Search Console Setup Guide for Sitaram Minerals

## üöÄ QUICK START

### Step 1: Add Property to Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property"
3. Choose "URL prefix" option
4. Enter: `https://sitaramminerals.com`
5. Click Continue

### Step 2: Verify Ownership
Choose one of these methods:

#### Option A: HTML File (Recommended)
1. Google will provide an HTML file
2. Download the file
3. Upload to `/public/` folder
4. Google will verify automatically
5. You can delete the file after verification

#### Option B: HTML Meta Tag
1. Copy the meta tag provided by Google
2. Add to `<head>` section of index.html
3. Google will verify after a few minutes

#### Option C: Domain Name Provider
1. If your domain is registered elsewhere
2. Google provides DNS TXT record instructions
3. Add it through your domain provider's DNS settings

### Step 3: Verify Sitemap
After property verification:

1. Go to **Sitemaps** in left menu
2. Click "Add a new sitemap"
3. Enter: `sitemap.xml`
4. Google will crawl and index your pages
5. You should see indexed pages within 24-48 hours

### Step 4: Monitor Performance
After 1-2 weeks:
1. Go to **Performance** section
2. You'll see:
   - Total clicks from search results
   - Search impressions
   - Click-through rate (CTR)
   - Average ranking position

### Step 5: Check Indexation
1. Go to **Coverage** section
2. Look for:
   - Valid pages (‚úÖ good)
   - Valid with warnings (‚ö†Ô∏è may need fixes)
   - Excluded pages (should include only intended pages)
   - Error pages (‚ùå fix these)

### Step 6: Submit URL for Indexing
To speed up indexing:
1. Click the search bar at top
2. Paste URL: `https://sitaramminerals.com`
3. Click "Request Indexing"
4. Google will crawl and index within minutes/hours
